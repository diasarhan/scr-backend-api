\# SCR Backend API



This project is a backend REST API built using ASP.NET Core and C#. It was developed as a final-year graduation project, focusing on backend development, RESTful API design, and database integration.



The system follows a layered architecture approach including Controllers, Services, Data access, and Models.



\## Project Purpose



This project was created as part of my final-year university graduation project to demonstrate practical skills in backend development, software architecture, and API design.



\## Features



\- RESTful API built with ASP.NET Core

\- CRUD operations (Create, Read, Update, Delete)

\- MySQL database integration

\- Layered architecture (Controllers, Services, Data, Models)

\- DTOs for structured data transfer

\- Swagger for API testing and documentation



\## Technologies Used



\- C#

\- ASP.NET Core

\- Entity Framework Core

\- MySQL

\- Swagger / OpenAPI



\## Project Structure



\- Controllers → Handles HTTP requests

\- Services → Business logic layer

\- Data → Database context and configuration

\- Models → Database entities

\- DTOs → Data transfer objects

\- Migrations → Database schema changes
